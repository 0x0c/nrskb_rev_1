
# Keyboard Layouter Playground

A sample circuit and netlist to quickly try the [Keyboard Layouter](https://github.com/yskoht/keyboard-layouter) plugin.

The sample JSON files are follow:

- ANSI 104
- ISO 105
- ErgoDox
- Atreus
